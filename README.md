# kudhru.github.io — Personal Website

Personal one-pager for Dhruv Kumar. All content lives in `index.html`.

## Adding a New Paper Entry

When a new paper is accepted, update the following in `index.html`:

### 1. Publications count (required)

**Location:** `index.html` — inside the `stats-strip` div, the first `stat-item`.

```html
<div class="stat-number">57+</div>
<div class="stat-label">Publications</div>
```

Increment the number by 1 (e.g., `57+` → `58+`).

---

That's the only change needed on this site. Detailed publication listings are maintained separately on the [PhantomGrad lab website](https://phantomgrad.github.io/research.html).
